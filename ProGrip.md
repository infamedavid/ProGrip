# ProGrip – Smart Object Grabbing for Blender

ProGrip is a lightweight Blender addon designed to simplify one of the most common tasks in character animation: **making characters grab and release objects**.

Instead of manually managing parenting, offsets, and animation fixes, ProGrip provides a fast and intuitive workflow that lets you **attach objects to animated rigs precisely at the moment of contact**.

Perfect for scenes where characters interact with props such as tools, weapons, cups, or any handheld object.

---

## Key Features

### One-click grab setup
Create a clean attachment between a bone and an object exactly when the grab happens.

### Animation-friendly workflow
Works naturally with existing animation. You animate first, then ProGrip handles the attachment.

### Precise contact alignment
Objects follow the animated bone perfectly from the moment the grab occurs.

### Clean release system
Detach objects smoothly when the character lets go.

### Non-destructive
Does not interfere with your rig or animation workflow.

### Designed for animators
Minimal setup, predictable behavior, and fast iteration.

---

## Typical Workflow

1. Animate the character approaching the object.  
2. Choose the bone that will grab the object (for example, a hand bone).  
3. Trigger the grab at the frame where contact happens.  
4. The object follows the animation naturally from that point.  
5. Release it whenever needed.
